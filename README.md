## Problem Statement
Long queues in college mess cause time wastage and overcrowding.

## Solution
Mess-Ease is a Google AppSheet‚Äìbased application designed to reduce overcrowding and long queues in college mess facilities by using AI-driven demand prediction. It 
- Collects mess attendance
- Predicts crowd levels
- Displays queue status in real time

## Tech Stack
- Google AppSheet 
- Google Sheets (Database)
- Google Apps Script (Automation & AI Integration)
- Gemini AI (Prediction Logic)

## How It Works
1. Mess staff enters daily meal count
2. Data is stored in Google Sheets
3. AI logic predicts crowd load
4. Queue status is updated automatically
5. Students view live queue status

## Features
- Admin mess entry
- AI prediction
- Queue status (Low / Medium / High)
- Live updates
